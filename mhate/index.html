m-hate.js works automatically out-of-the-box, but has an api in case it breaks your website or you have dynamic elements that need to be updated. If you place the script tag in the head of your HTML, you can use the following commands to control the script:

mHate.excludeTags(list) and mHate.excludeClasses(list)

If purgery breaks your website, you can use these commands to prevent m-hate from interfereng with specific elements. Just pass a list of tag names or class names and m-hate will ignore them. Tag names must be passed in uppercase. Default tags excluded: 'SCRIPT', 'STYLE', 'TEXTAREA', 'INPUT', 'LABEL', 'PRE', 'HATE'

mHate.crossAllM()

Purges the entire body of your webpage. Runs automatically on page load

mHate.crossIn(element)

Purges given HTML element and all its children that are not in exclude lists
